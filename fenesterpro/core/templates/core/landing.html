No. The workflow is useful for both growing workshops and larger operations that handle multiple concurrent sites.

How quickly can I start?

Immediately. Create a project, add windows, and generate reports in the same dashboard session.

Can the team review old projects?

Yes. Project history is available through the project list and detail pages for follow-up and revisions.
